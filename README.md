Official repository for ArtHOI: Taming Foundation Models for Monocular 4D Reconstruction of Hand-Articulated-Object Interactions.

## Overview

ArtHOI targets 4D reconstruction of hand-articulated-object interactions under monocular video input, without requiring object pre-scans or multi-view capture.
